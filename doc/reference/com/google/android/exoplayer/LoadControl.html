Public Methods

public abstract Allocator getAllocator ()

Gets the Allocator that loaders should use to obtain memory allocations into which data can be loaded.

Returns
  • The Allocator to use.

public abstract void register ( loader, int bufferSizeContribution)

Registers a loader.

Parameters
loader The loader being registered.
bufferSizeContribution For controls whose Allocators maintain a pool of memory for the purpose of satisfying allocation requests, this is a hint indicating the loader's desired contribution to the size of the pool, in bytes.

public abstract void trimAllocator ()

Hints to the control that it should consider trimming any unused memory being held in order to satisfy allocation requests.

This method is typically invoked by a recently unregistered loader, once it has released all of its allocations back to the Allocator.

public abstract void unregister ( loader)

Unregisters a loader.

Parameters
loader The loader being unregistered.

public abstract boolean update ( loader, long playbackPositionUs, long nextLoadPositionUs, boolean loading, boolean failed)

Invoked by a loader to update the control with its current state.

This method must be called by a registered loader whenever its state changes. This is true even if the registered loader does not itself wish to start its next load (since the state of the loader will still affect whether other registered loaders are allowed to proceed).

Parameters
loader The loader invoking the update.
playbackPositionUs The loader's playback position.
nextLoadPositionUs The loader's next load position. -1 if finished, failed, or if the next load position is not yet known.
loading Whether the loader is currently loading data.
failed Whether the loader has failed.
Returns
  • True if the loader is allowed to start its next load. False otherwise.
