public long getBufferedPositionUs ()

Returns an estimate of the position up to which data is buffered.

This method should not be called until after the extractor has been successfully prepared.

Returns
  • An estimate of the absolute position in microseconds up to which data is buffered, or END_OF_TRACK_US if data is buffered to the end of the stream, or UNKNOWN_TIME_US if no estimate is available.

public int readSample (int track, SampleHolder sampleHolder)

Reads the next sample in the track at index track into sampleHolder, returning SAMPLE_READ if it is available.

Advances to the next sample if a sample was read.

Parameters
track The index of the track from which to read a sample.
sampleHolder The holder for read sample data, if SAMPLE_READ is returned.
Returns
  • SAMPLE_READ if a sample was read into sampleHolder, or END_OF_STREAM if the last samples in all tracks have been read, or NOTHING_READ if the sample cannot be read immediately as it is not loaded.
