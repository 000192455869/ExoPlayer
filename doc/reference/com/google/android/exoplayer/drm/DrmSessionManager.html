Constants

public static final int STATE_CLOSED

The session is closed.

Constant Value: 1 (0x00000001)

public static final int STATE_ERROR

The error state. getError() can be used to retrieve the cause.

Constant Value: 0 (0x00000000)

public static final int STATE_OPENED

The session is open, but does not yet have the keys required for decryption.

Constant Value: 3 (0x00000003)

public static final int STATE_OPENED_WITH_KEYS

The session is open and has the keys required for decryption.

Constant Value: 4 (0x00000004)

public static final int STATE_OPENING

The session is being opened (i.e. open(Map, String) has been called, but the session is not yet open).

Constant Value: 2 (0x00000002)

Public Methods

public abstract void close ()

Closes the session.

public abstract Exception getError ()

Gets the cause of the error state.

This method may be called when the manager is in any state.

Returns
  • An exception if the state is STATE_ERROR. Null otherwise.

public abstract MediaCrypto getMediaCrypto ()

Gets a MediaCrypto for the open session.

This method may be called when the manager is in the following states: STATE_OPENED, STATE_OPENED_WITH_KEYS

Returns
  • A MediaCrypto for the open session.
Throws
IllegalStateException If called when a session isn't opened.

public abstract int getState ()

Gets the current state of the session.

Returns
  • One of STATE_ERROR, STATE_CLOSED, STATE_OPENING, STATE_OPENED and STATE_OPENED_WITH_KEYS.

public abstract void open (<UUID, byte[]> drmInitData, mimeType)

Opens the session, possibly asynchronously.

Parameters
drmInitData Initialization data for the drm schemes supported by the media, keyed by scheme UUID.
mimeType The mimeType of the media.

public abstract boolean requiresSecureDecoderComponent ( mimeType)

Whether the session requires a secure decoder for the specified mime type.

Normally this method should return requiresSecureDecoderComponent(String), however in some cases implementations may wish to modify the return value (i.e. to force a secure decoder even when one is not required).

This method may be called when the manager is in the following states: STATE_OPENED, STATE_OPENED_WITH_KEYS

Returns
  • Whether the open session requires a secure decoder for the specified mime type.
Throws
IllegalStateException If called when a session isn't opened.
