SampleSource

com.google.android.exoplayer.SampleSource
Known Indirect Subclasses
ChunkSampleSource, DefaultSampleSource, HlsSampleSource
ChunkSampleSource A SampleSource that loads media in Chunks, which are themselves obtained from a ChunkSource. 
DefaultSampleSource SampleSource that extracts sample data using a SampleExtractor  
HlsSampleSource A SampleSource for HLS streams. 

Class Overview

A source of media samples.

A SampleSource may expose one or multiple tracks. The number of tracks and information about each can be queried using getTrackCount() and getTrackInfo(int) respectively.

public static final int DISCONTINUITY_READ

A discontinuity in the sample stream.

Constant Value: -5 (0xfffffffb)

public static final int END_OF_STREAM

The end of stream has been reached.

Constant Value: -1 (0xffffffff)

public static final int FORMAT_READ

A format was read.

Constant Value: -4 (0xfffffffc)

public static final int NOTHING_READ

Neither a sample nor a format was read in full. This may be because insufficient data is buffered upstream. If multiple tracks are enabled, this return value may indicate that the next piece of data to be returned from the SampleSource corresponds to a different track than the one for which data was requested.

Constant Value: -2 (0xfffffffe)

public static final int SAMPLE_READ

A sample was read.

Constant Value: -3 (0xfffffffd)

Public Methods

public abstract boolean continueBuffering (long positionUs)

Indicates to the source that it should still be buffering data.

Parameters
positionUs The current playback position.
Returns
  • True if the source has available samples, or if the end of the stream has been reached. False if more data needs to be buffered for samples to become available.
Throws
IOException If an error occurred reading from the source.

public abstract void disable (int track)

Disable the specified track.

This method should not be called until after the source has been successfully prepared.

Parameters
track The track to disable.

public abstract void enable (int track, long positionUs)

Enable the specified track. This allows the track's format and samples to be read from readData(int, long, MediaFormatHolder, SampleHolder, boolean).

This method should not be called until after the source has been successfully prepared.

Parameters
track The track to enable.
positionUs The player's current playback position.

public abstract long getBufferedPositionUs ()

Returns an estimate of the position up to which data is buffered.

This method should not be called until after the source has been successfully prepared.

Returns
  • An estimate of the absolute position in microseconds up to which data is buffered, or END_OF_TRACK_US if data is buffered to the end of the stream, or UNKNOWN_TIME_US if no estimate is available.

public abstract int getTrackCount ()

Returns the number of tracks exposed by the source.

Returns
  • The number of tracks.

public abstract TrackInfo getTrackInfo (int track)

Returns information about the specified track.

This method should not be called until after the source has been successfully prepared.

Returns
  • Information about the specified track.

public abstract boolean prepare ()

Prepares the source.

Preparation may require reading from the data source (e.g. to determine the available tracks and formats). If insufficient data is available then the call will return false rather than block. The method can be called repeatedly until the return value indicates success.

Returns
  • True if the source was prepared successfully, false otherwise.
Throws
IOException If an error occurred preparing the source.

public abstract int readData (int track, long positionUs, MediaFormatHolder formatHolder, SampleHolder sampleHolder, boolean onlyReadDiscontinuity)

Attempts to read either a sample, a new format or or a discontinuity from the source.

This method should not be called until after the source has been successfully prepared.

Note that where multiple tracks are enabled, NOTHING_READ may be returned if the next piece of data to be read from the SampleSource corresponds to a different track than the one for which data was requested.

Parameters
track The track from which to read.
positionUs The current playback position.
formatHolder A MediaFormatHolder object to populate in the case of a new format.
sampleHolder A SampleHolder object to populate in the case of a new sample. If the caller requires the sample data then it must ensure that data references a valid output buffer.
onlyReadDiscontinuity Whether to only read a discontinuity. If true, only DISCONTINUITY_READ or NOTHING_READ can be returned.
Returns
  • The result, which can be SAMPLE_READ, FORMAT_READ, DISCONTINUITY_READ, NOTHING_READ or END_OF_STREAM.
Throws
IOException If an error occurred reading from the source.

public abstract void release ()

Releases the SampleSource.

Seeks to the specified time in microseconds.

This method should not be called until after the source has been successfully prepared.

Parameters
positionUs The seek position in microseconds.
