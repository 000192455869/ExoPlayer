Class Overview

Default version of an extractor to facilitate data retrieval from the WebM container format.

WebM is a subset of the EBML elements defined for Matroska. More information about EBML and Matroska is available here. More info about WebM is here.

public boolean read (NonBlockingInputStream inputStream, SampleHolder sampleHolder)

Consumes data from a NonBlockingInputStream.

If the return value is false, then a sample may have been partially read into sampleHolder. Hence the same SampleHolder instance must be passed in subsequent calls until the whole sample has been read.

Parameters
inputStream The input stream from which data should be read
sampleHolder A SampleHolder into which the sample should be read
Returns
  • true if a sample has been read into the sample holder

public boolean seekTo (long seekTimeUs, boolean allowNoop)

Seeks to a position before or equal to the requested time.

Parameters
seekTimeUs The desired seek time in microseconds
allowNoop Allow the seek operation to do nothing if the seek time is in the current segment, is equal to or greater than the time of the current sample, and if there does not exist a sync frame between these two times
Returns
  • True if the operation resulted in a change of state. False if it was a no-op
