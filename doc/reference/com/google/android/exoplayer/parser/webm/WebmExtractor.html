Class Overview

Facilitates the extraction of data from the WebM container format with a non-blocking, incremental parser based on EbmlReader.

WebM is a subset of the EBML elements defined for Matroska. More information about EBML and Matroska is available here. More info about WebM is here.

protected boolean onBinaryElement (NonBlockingInputStream inputStream, int id, long elementOffset, int headerSize, int contentsSize)

Called when a binary element is encountered in the NonBlockingInputStream. The element header (containing element ID and content size) will already have been read. Subclasses must exactly read the entire contents of the element, which is contentsSize bytes in length. It's guaranteed that the full element contents will be immediately available from inputStream.

Several methods are available for reading the contents of a binary element:

  • readVarint(NonBlockingInputStream).
  • readBytes(NonBlockingInputStream, byte[], int).
  • readBytes(NonBlockingInputStream, ByteBuffer, int).
  • skipBytes(NonBlockingInputStream, int).
  • getBytesRead().

Parameters
inputStream The NonBlockingInputStream from which this element's contents should be read.
id The integer ID of this element.
elementOffset The byte offset where this element starts.
headerSize The byte length of this element's ID and size header.
contentsSize The byte length of this element's contents.
Returns
  • true if parsing should continue or false if it should stop right away.
