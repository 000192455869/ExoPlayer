Class Overview

A component of an ExoPlayer that can receive messages on the playback thread.

Messages can be delivered to a component via sendMessage(ExoPlayer.ExoPlayerComponent, int, Object) and blockingSendMessage(ExoPlayer.ExoPlayerComponent, int, Object).
