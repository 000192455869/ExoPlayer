ExoPlayer.Listener

com.google.android.exoplayer.ExoPlayer.Listener

Class Overview

Interface definition for a callback to be notified of changes in player state.

Summary

Public Methods

public abstract void onPlayWhenReadyCommitted ()

Invoked when the current value of getPlayWhenReady() has been reflected by the internal playback thread.

An invocation of this method will shortly follow any call to setPlayWhenReady(boolean) that changes the state. If multiple calls are made in rapid succession, then this method will be invoked only once, after the final state has been reflected.

public abstract void onPlayerError (ExoPlaybackException error)

Invoked when an error occurs. The playback state will transition to STATE_IDLE immediately after this method is invoked. The player instance can still be used, and release() must still be called on the player should it no longer be required.

Parameters
error The error.

public abstract void onPlayerStateChanged (boolean playWhenReady, int playbackState)

Invoked when the value returned from either getPlayWhenReady() or getPlaybackState() changes.

Parameters
playWhenReady Whether playback will proceed when ready.
playbackState One of the STATE constants defined in this class.
