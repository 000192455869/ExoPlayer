Public Constructors

public WebvttSubtitle ( cueText, long startTimeUs, long[] cueTimesUs)

Parameters
cueText Text to be displayed during each cue.
startTimeUs The start time of the subtitle.
cueTimesUs Cue event times, where cueTimesUs[2 * i] and cueTimesUs[(2 * i) + 1] are the start and end times, respectively, corresponding to cueText[i].

public long getLastEventTime ()

Convenience method for obtaining the last event time.

Returns
  • The time of the last event in microseconds, or -1 if getEventTimeCount() == 0.

public int getNextEventTimeIndex (long timeUs)

Gets the index of the first event that occurs after a given time (exclusive).

Parameters
timeUs The time in microseconds.
Returns
  • The index of the next event, or -1 if there are no events after the specified time.

public long getStartTime ()

Gets the start time of the subtitle.

Note that the value returned may be less than getEventTime(0), since a subtitle may begin prior to the time of the first event.

Returns
  • The start time of the subtitle in microseconds.

Parameters
timeUs The time in microseconds.
Returns
