Public Constructors

public RangedUri (Uri baseUri, String stringUri, long start, long length)

Constructs an ranged uri.

The uri is built according to the following rules:

  • If baseUri is null or if stringUri is absolute, then baseUri is ignored and the url consists solely of stringUri.
  • If stringUri is null, then the url consists solely of baseUrl.
  • Otherwise, the url consists of the concatenation of baseUri and stringUri.

Parameters
baseUri An uri that can form the base of the uri defined by the instance.
stringUri A relative or absolute uri in string form.
start The (zero based) index of the first byte of the range.
length The length of the range, or -1 to indicate that the range is unbounded.

Public Methods

public RangedUri attemptMerge (RangedUri other)

Attempts to merge this RangedUri with another.

A merge is successful if both instances define the same Uri, and if one starte the byte after the other ends, forming a contiguous region with no overlap.

If other is null then the merge is considered unsuccessful, and null is returned.

Parameters
other The RangedUri to merge.
Returns
  • The merged RangedUri if the merge was successful. Null otherwise.
