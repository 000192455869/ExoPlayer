public abstract T parse ( inputStream, inputEncoding, contentId, baseUri)

Parses a manifest from an InputStream.

Parameters
inputStream The input stream to consume.
inputEncoding The encoding of the input stream. May be null if the input encoding is unknown.
contentId The content id to which the manifest corresponds. May be null.
baseUri If the manifest contains relative uris, this is the uri they are relative to. May be null.
Returns
  • The parsed manifest.
Throws
IOException If an error occurs reading the data.
ParserException If an error occurs parsing the data.
