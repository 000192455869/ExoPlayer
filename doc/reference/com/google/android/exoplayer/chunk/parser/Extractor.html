Public Methods

public abstract long getDurationUs ()

Returns the duration of the stream in microseconds, or UNKNOWN_TIME_US if unknown.

public abstract MediaFormat getFormat ()

Returns the format of the samples contained within the media stream.

Returns
  • The sample media format, or null if the format has yet to be parsed.

public abstract SegmentIndex getIndex ()

Returns the segment index parsed from the stream.

Returns
  • The segment index, or null if a SIDX atom has yet to be parsed.

public abstract Map<UUID, byte[]> getPsshInfo ()

Returns the pssh information parsed from the stream.

Returns
  • The pssh information. May be null if pssh data has yet to be parsed, or if the stream does not contain any pssh data.

public abstract boolean hasRelativeIndexOffsets ()

Returns true if the offsets in the index returned by getIndex() are relative to the first byte following the initialization data, or false if they are absolute (i.e. relative to the first byte of the stream).

Returns
  • True if the offsets are relative to the first byte following the initialization data. False otherwise.

public abstract int read (NonBlockingInputStream inputStream, SampleHolder out)

Consumes data from a NonBlockingInputStream.

The read terminates if the end of the input stream is reached, if an attempt to read from the input stream returned 0 bytes of data, or if a sample is read. The returned flags indicate both the reason for termination and data that was parsed during the read.

Parameters
inputStream The input stream from which data should be read.
out A SampleHolder into which the next sample should be read. If null then RESULT_NEED_SAMPLE_HOLDER will be returned once a sample has been reached.
Returns
  • One or more of the RESULT_* flags defined in this class.
Throws
ParserException If an error occurs parsing the media data.

public abstract boolean seekTo (long seekTimeUs, boolean allowNoop)

Seeks to a position before or equal to the requested time.

Parameters
seekTimeUs The desired seek time in microseconds.
allowNoop Allow the seek operation to do nothing if the seek time is in the current fragment run, is equal to or greater than the time of the current sample, and if there does not exist a sync frame between these two times.
Returns
  • True if the operation resulted in a change of state. False if it was a no-op.
