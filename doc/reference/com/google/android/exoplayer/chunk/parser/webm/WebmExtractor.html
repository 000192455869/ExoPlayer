Class Overview

An extractor to facilitate data retrieval from the WebM container format.

WebM is a subset of the EBML elements defined for Matroska. More information about EBML and Matroska is available here. More info about WebM is here.

public int read (NonBlockingInputStream inputStream, SampleHolder sampleHolder)

Consumes data from a NonBlockingInputStream.

The read terminates if the end of the input stream is reached, if an attempt to read from the input stream returned 0 bytes of data, or if a sample is read. The returned flags indicate both the reason for termination and data that was parsed during the read.

Parameters
inputStream The input stream from which data should be read.
sampleHolder A SampleHolder into which the next sample should be read. If null then RESULT_NEED_SAMPLE_HOLDER will be returned once a sample has been reached.
Returns
  • One or more of the RESULT_* flags defined in this class.
Throws
ParserException

Seeks to a position before or equal to the requested time.

Parameters
seekTimeUs The desired seek time in microseconds.
allowNoop Allow the seek operation to do nothing if the seek time is in the current fragment run, is equal to or greater than the time of the current sample, and if there does not exist a sync frame between these two times.
Returns
  • True if the operation resulted in a change of state. False if it was a no-op.
