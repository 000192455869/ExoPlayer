public abstract void onConsumptionError (int sourceId, IOException e)

Invoked when an error occurs consuming loaded data.

Parameters
sourceId The id of the reporting SampleSource.
e The cause of the failure.

public abstract void onDownstreamDiscarded (int sourceId, int mediaStartTimeMs, int mediaEndTimeMs, long bytesDiscarded)

Invoked when data is removed from the front of the buffer, typically due to a seek or because the data has been consumed.

Parameters
sourceId The id of the reporting SampleSource.
mediaStartTimeMs The media time of the start of the discarded data.
mediaEndTimeMs The media time of the end of the discarded data.
bytesDiscarded The length of the data being discarded in bytes.

public abstract void onDownstreamFormatChanged (int sourceId, String formatId, int trigger, int mediaTimeMs)

Invoked when the downstream format changes (i.e. when the format being supplied to the caller of readData(int, long, MediaFormatHolder, SampleHolder, boolean) changes).

Parameters
sourceId The id of the reporting SampleSource.
formatId The format id.
trigger The trigger specified in the corresponding upstream load, as specified by the ChunkSource.
mediaTimeMs The media time at which the change occurred.

public abstract void onLoadCanceled (int sourceId, long bytesLoaded)

Invoked when the current upstream load operation is canceled.

Parameters
sourceId The id of the reporting SampleSource.
bytesLoaded The number of bytes that were loaded prior to the cancellation.

public abstract void onLoadCompleted (int sourceId, long bytesLoaded)

Invoked when the current load operation completes.

Parameters
sourceId The id of the reporting SampleSource.
bytesLoaded The number of bytes that were loaded.

public abstract void onLoadStarted (int sourceId, String formatId, int trigger, boolean isInitialization, int mediaStartTimeMs, int mediaEndTimeMs, long length)

Invoked when an upstream load is started.

Parameters
sourceId The id of the reporting SampleSource.
formatId The format id.
trigger A trigger for the format selection, as specified by the ChunkSource.
isInitialization Whether the load is for format initialization data.
mediaStartTimeMs The media time of the start of the data being loaded, or -1 if this load is for initialization data.
mediaEndTimeMs The media time of the end of the data being loaded, or -1 if this load is for initialization data.
length The length of the data being loaded in bytes, or LENGTH_UNBOUNDED if the length of the data has not yet been determined.

public abstract void onUpstreamDiscarded (int sourceId, int mediaStartTimeMs, int mediaEndTimeMs, long bytesDiscarded)

Invoked when data is removed from the back of the buffer, typically so that it can be re-buffered using a different representation.

Parameters

mediaStartTimeMs The media time of the start of the discarded data.
mediaEndTimeMs The media time of the end of the discarded data.
bytesDiscarded The length of the data being discarded in bytes.
