Parameters
dataSource A DataSource for loading the data.
dataSpec Defines the data to be loaded.
format The format of the stream to which this chunk belongs.
trigger The reason for this chunk being selected.
startTimeUs The start time of the media contained by the chunk, in microseconds.
endTimeUs The end time of the media contained by the chunk, in microseconds.
nextChunkIndex The index of the next chunk, or -1 if this is the last chunk.
sampleFormat The format of the media contained by the chunk.
headerData Custom header data for the sample. May be null. If set, the header data is prepended to the sample data returned when read(SampleHolder) is called. It is not reflected in the values returned by bytesLoaded() and getLength().

public boolean prepare ()

Prepares the chunk for reading. Does nothing if the chunk is already prepared.

Preparation may require consuming some of the chunk. If the data is not yet available then this method will return false rather than block. The method can be called repeatedly until the return value indicates success.

Returns
  • True if the chunk was prepared. False otherwise.

public boolean read (SampleHolder holder)

Reads the next media sample from the chunk.

Parameters
holder A holder to store the read sample.
Returns

public boolean sampleAvailable ()

public boolean seekTo (long positionUs, boolean allowNoop)

Seeks to the specified position within the chunk.

Parameters
positionUs The desired seek time in microseconds.
allowNoop True if the seek is allowed to do nothing if the result is more accurate than seeking to a key frame. Always pass false if it is required that the next sample be a key frame.
Returns
  • True if the seek results in a discontinuity in the sequence of samples returned by read(SampleHolder). False otherwise.
