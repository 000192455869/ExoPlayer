package

com.google.android.exoplayer.upstream

Interfaces

Allocation An Allocation, defined to consist of a set of fragments of underlying byte arrays. 
Allocator A source of Allocations. 
BandwidthMeter Provides estimates of the currently available bandwidth. 
DataSink A component that consumes media data. 
DataSource A component that provides media data. 
DefaultBandwidthMeter.EventListener Interface definition for a callback to be notified of DefaultBandwidthMeter events. 
Loader.Callback Interface definition for a callback to be notified of Loader events. 
Loader.Loadable Interface definition of an object that can be loaded using a Loader. 
NonBlockingInputStream Represents a source of bytes that can be consumed by downstream components. 
TransferListener Interface definition for a callback to be notified of data transfer events. 

Classes

BufferPool An Allocator that maintains a pool of fixed length byte arrays (buffers). 
ByteArrayDataSink A DataSink for writing to a byte array. 
ByteArrayDataSource A DataSource for reading from a byte array. 
ByteArrayNonBlockingInputStream An implementation of NonBlockingInputStream for reading data from a byte array. 
DataSourceInputStream Allows data corresponding to a given DataSpec to be read from a DataSource and consumed as an InputStream. 
DataSourceStream Loads data from a DataSource into an in-memory Allocation. 
DataSpec Defines a region of media data. 
DefaultBandwidthMeter Counts transferred bytes while transfers are open and creates a bandwidth sample and updated bandwidth estimate each time a transfer ends. 
FileDataSource A local file DataSource. 
HttpDataSource An http DataSource. 
Loader Manages the background loading of Loader.Loadables. 
NetworkLock A network task prioritization mechanism. 
PriorityDataSource Allows open(DataSpec) and read(byte[], int, int) calls only if the specified priority is the highest priority of any task. 
TeeDataSource Tees data into a DataSink as the data is read. 
UriDataSource A data source that fetches data from a local or remote DataSpec. 

Exceptions

DataSourceStream.DataSourceStreamLoadException Thrown when an error is encountered trying to load data into a DataSourceStream. 
FileDataSource.FileDataSourceException Thrown when IOException is encountered during local file read operation. 
HttpDataSource.HttpDataSourceException Thrown when an error is encountered when trying to read from HTTP data source. 
HttpDataSource.InvalidContentTypeException Thrown when the content type is invalid. 
HttpDataSource.InvalidResponseCodeException Thrown when an attempt to open a connection results in a response code not in the 2xx range. 
Loader.UnexpectedLoaderException Thrown when an unexpected exception is encountered during loading. 
NetworkLock.PriorityTooLowException Thrown when a task is attempts to proceed when it does not have the highest priority. 
UnexpectedLengthException Thrown when the length of some data does not match an expected length.
