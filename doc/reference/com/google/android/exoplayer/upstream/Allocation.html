Allocation

com.google.android.exoplayer.upstream.Allocation

Class Overview

An Allocation, defined to consist of a set of fragments of underlying byte arrays.

The byte arrays in which the fragments are located are obtained by getBuffers(). For each, the offset and length of the fragment within the byte array are obtained using getFragmentOffset(int) and getFragmentLength(int) respectively.

public abstract void ensureCapacity (int size)

Ensures the allocation has a capacity greater than or equal to the specified size in bytes.

If size is greater than the current capacity of the allocation, then it will grow to have a capacity of at least size. The allocation is grown by adding new fragments. Existing fragments remain unchanged, and any data that has been written to them will be preserved.

If size is less than or equal to the capacity of the allocation, then the call is a no-op.

size The minimum required capacity, in bytes.
