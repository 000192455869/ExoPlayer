Fields

public final long absoluteStreamPosition

The absolute position of the data in the full stream.

public final String key

A key that uniquely identifies the original stream. Used for cache indexing. May be null if the DataSpec is not intended to be used in conjunction with a cache.

public final long length

The length of the data. Greater than zero, or equal to LENGTH_UNBOUNDED.

public final long position

The position of the data when read from uri. Always equal to absoluteStreamPosition if uriIsFullStream.

public final Uri uri

Identifies the source from which data should be read.

public final boolean uriIsFullStream

True if the data at uri is the full stream. False otherwise. An example where this may be false is if uri defines the location of a cached part of the stream.
