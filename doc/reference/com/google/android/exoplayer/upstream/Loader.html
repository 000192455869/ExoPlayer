public void startLoading ( looper, Loader.Loadable loadable, Loader.Callback callback)

Start loading a Loader.Loadable.

A Loader instance can only load one Loader.Loadable at a time, and so this method must not be called when another load is in progress.

Parameters
looper The looper of the thread on which the callback should be invoked.
loadable The Loader.Loadable to load.
callback A callback to invoke when the load ends.
